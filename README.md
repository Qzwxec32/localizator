# localizator
Simple mod for translation text to your language / Простой мод для перевода текста на ваш язык
ENG:
After installation, you need to launch the game. 
On the first run, a file called translation.txt will be created in the bepinex/config folder.

In this file, you write the text you want to translate, followed by = and the translated version you want to get.

Example:

Bag=Сумка
Backpack=Рюкзак
Beast Horn=Рог зверя
Discovered a new location=Открыта новая локация
Wait Here=Ждать
Resume=Следовать

Rus:
После установки нужно зайти в игру, при первом входе в папке bepinex/config создается файл translation.txt 
Туда пишем текст который хотим перевести и через символ = текст который хотим получить 
Пример: 
Bag=Сумка 
Backpack=Рюкзак 
Beast Horn=Рог зверя 
Discovered a new location=Открыта новая локация 
Wait Here=Ждать 
Resume=Следовать
